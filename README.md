# Automation Scripts Extensions

> [!Note]
> Automation scripts should be packaged as `org.aegisub.Aegisub.Automations.*`. Please ensure unique file names to avoid conflicts.

Because FlatHub Team is refusing to have Aegisub scripts packaged separately (https://github.com/flathub/flathub/pull/6859#issuecomment-3217463254), we have them bundled and packaged as separate extensions that can be manually installed.
